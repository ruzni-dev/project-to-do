# Project To-Do

A responsive to-do list web app built with vanilla HTML, CSS, and JavaScript. It supports task creation, completion tracking, inline filtering, drag-and-drop reordering, dark mode, and editing tasks through a modal.

## Features

- Add tasks with an optional due date.
- Mark tasks as complete or incomplete.
- Edit tasks in a dedicated modal dialog.
- Delete tasks.
- Reorder tasks by dragging them.
- Filter tasks by all, active, or completed.
- Toggle between light and dark theme.
- Save tasks and theme preference in local storage.
- Works on desktop and mobile layouts.

## How To Run

1. Open the project folder in VS Code or your file explorer.
2. Open `index.html` in a browser.
3. Start adding tasks from the input box.

No build step or dependencies are required.

## Usage Notes

- Click `Add` to create a task.
- Click `Edit` to open the modal and update task text or due date.
- Drag task cards to change their order.
- Use `Dark Mode` to switch themes.
- Use the filter buttons to view specific task states.
